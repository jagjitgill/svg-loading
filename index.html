Implementation:

  1. Add the CSS and JS files.
  2. Add svg animation tempates in the html file as required (Circular, circle or/and infinity).
  3. Build loading element:
    var circularLoading = initializeLoading('.button-class');
  4. Success/Fail
    circularLoading.triggerFail(); OR circularLoading.triggerSuccess();
    The default state messages is null. You can pass custom values via 'data' attributes. The success/fail message can also be passed directly in the call.
    → loading1.triggerSuccess('All Done!'); → loading1.triggerFail('Action Failed :(');
  5. Available custom options.
    { type: '', position: '', scale: '' }
    type → circular, circle, infinity
    position → top, bottom, left, right and inline (inline for p, div, span etc.)
    scale → x1-5, x2, x2-5 ... x4 (For inline only)

    ex: var loading = initializeLoading('.my-class', { type: 'circular', position: 'inline', scale: 'x1' });
